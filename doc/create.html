# 创建项目

# 自动创建

控制台输入

 weexplus create

# 手动创建

如果下载很慢，请尝试手动下载

点击下载模板下载地址

解压boilerplate-master.zip

执行weexplus rename

注意windows下cmd命令要和boilerplate-master在一个盘符下

# 安装npm依赖

cd进入项目目录执行

npm install
在 GitHub 上编辑此页
