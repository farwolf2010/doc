# 运行原生端

开启一个控制台运行

再开启一个控制台运行

# 模拟器如何看效果

这里特指xcode,android的暂时不支持模拟器
模拟器无法扫码，所以没法像上面那样
首先修改configs/weexplus.json中debugEntry的值
值就是你选中js后二维码上方的地址
然后执行weexplus copy(未开启weexplus start的情况才要这步，已开启不需要)
然后运行原生项目
