# 快速开始

本章节旨在让大家快速体验weexplus的魅力，所以复杂的步骤我们都丢到后面再说

# 安装基础包

npm install weex-toolkit -g
npm install weexplus -g

# 创建项目

weexplus create

如下图

控制台cd进入新建的项目目录执行

npm install

安装依赖，不报错的话，项目的前端环境就好了

# 启动项目

接下来开启编译执行

npm run native

再开启一个控制台，同样cd进入项目目录执行

weexplus start

此时会自动弹出浏览器，界面如下 选中index.js,二维码会重新生成

# 手机看效果

下载开发app

安装完毕打开app，点设置->扫描二维码，扫码网页上的二维码

好了，现在就能看到界面了，到这里就可以开始开发了 试试改变src/index.vue的代码，保存之后界面会自动刷新
